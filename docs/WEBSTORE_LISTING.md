## Short Description

中文：
在长对话页面中自动生成可点击目录，快速跳转定位消息，并支持多选导出 Markdown。

English:
Auto-build a clickable sidebar index for long web conversations, with quick jump navigation and Markdown export.

## Detailed Description

中文：
Sinan（司南）帮助你在长对话页面中更快定位内容。插件会识别你的提问条目，在页面右侧生成悬浮目录，点击即可跳转到对应位置。  

核心能力：
- 自动提取用户提问并生成导航目录
- 当前阅读位置高亮（Scroll Spy）
- 侧边栏可折叠，减少页面干扰
- 支持多选对话并导出为单个 Markdown 文件
- 适配浅色与深色界面

适用场景：
- 快速回看多轮问答中的关键问题
- 从超长对话中定位历史上下文
- 将选中对话片段归档到本地知识库

English:
Sinan helps you navigate long web conversations efficiently. It detects your prompt turns and builds a floating sidebar index, so you can jump directly to the part you need.

Key features:
- Auto-extract prompt turns into a clickable index
- Scroll Spy highlights the active section while reading
- Collapsible sidebar to reduce visual noise
- Multi-select turns and export one Markdown file
- Light and dark theme adaptation
